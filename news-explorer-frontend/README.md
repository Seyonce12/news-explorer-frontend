News Explorer — Stage 1 (Vite scaffold)
=======================================

This project was prepared as a Stage 1 frontend scaffold for the News Explorer assignment.
It follows the asset structure you specified: all images start at `/assets/images/...` and fonts at `/assets/fonts/...`.

How to run:
1. npm install
2. npm run dev
3. Open http://localhost:5173

Git:
Create and switch to the branch you requested:
  git checkout -b stage-1-frontend-api

Notes:
- Images are placeholders (small 1x1 PNGs). Replace them with the real images from your assets folder.
- Fonts in assets/fonts are placeholders; replace them with files from your figma export.
- API calls are mocked in src/utils/api.js. Swap with real fetch later.
