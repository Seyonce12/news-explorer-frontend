# News Explorer

**Live on GitHub Pages:** https://seyonce12.github.io/news-explorer-frontend/

---

## About the App

News Explorer is a responsive React-based web application for discovering and saving news articles.  
You can search for the latest stories on any topic, sign up for an account, and keep a personal library of articles to revisit later.  

The interface is clean and adaptable to different devices, making it easy to browse, read, and manage your news from anywhere.

---

## Quick Start Guide

1. **Search News**
   - Type any topic into the search bar on the homepage.
   - Click **Search** to load matching news articles.

2. **Create an Account**
   - Click **Sign in** at the top right.
   - Choose **Sign up** and enter your details.
   - After registration, sign in to access saved articles.

3. **Save Articles**
   - While browsing, click the **bookmark icon** on any article to save it.
   - Your saved articles are stored in your account.

4. **View Saved Articles**
   - Go to the **Saved articles** section in the navigation.
   - Browse all articles you’ve saved.
   - Remove any by clicking the **trash icon**.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/news-explorer-frontend.git
   ```

2. Install dependencies:
   ```bash
   cd news-explorer-frontend
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
